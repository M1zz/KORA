# KORA Pages

KORA의 공식 지원/개인정보 처리방침 사이트.

배포: <https://m1zz.github.io/KORA/>

## GitHub Pages 활성화

GitHub 리포 → **Settings → Pages**:

- **Source**: `Deploy from a branch`
- **Branch**: `main` / `/docs`
- 저장 후 1~2분 뒤 URL 활성

## 페이지

- `index.html` — 앱 소개 / 홈
- `support.html` — 자주 묻는 질문 + 연락처
- `privacy.html` — 개인정보 처리방침
- `style.css` — 공통 스타일
- `lang.js` — 한국어/일본어/영어 토글
- `icon.png` — 앱 아이콘 (1024×1024)

## 언어 동작

- 처음 방문 시 브라우저 언어로 자동 선택 (ko / ja / en)
- 사용자가 토글하면 `localStorage`에 저장되어 다음 방문에도 유지
