자주 묻는 질문

저장한 장소가 안 보여요

아래 두 가지를 확인해주세요.

  • '가고 싶은' 탭의 우하단 리스트 보기 버튼을 눌렀나요? 기본 화면은 지도입니다.
  • 인스타그램에서 공유로 저장한 경우, 가게 이름을 직접 입력해야 저장이 완성됩니다.

출구 번호가 안 나와요

다음 두 경우엔 출구 안내가 표시되지 않을 수 있습니다.

  • 저장된 장소에 좌표가 없는 경우 (드물게 인스타 저장 직후) → 다음 앱 실행 시 자동 보강됩니다.
  • 도착역이 데이터셋(약 325개 역)에 없는 경우 → 외곽 일부 역은 미커버. 알려주시면 추가 예정입니다.

다른 언어로 바꾸고 싶어요

전철 탭의 환영 화면이나 역 이름을 길게 누르면 언어 선택 시트가 열립니다 (한국어 · 日本語 · English · 中文).

스크린샷에서 장소 추출이 안 돼요

이미지에 한국어 / 일본어 / 영어 / 중국어 텍스트가 명확히 있어야 합니다. 텍스트가 너무 작거나 흐린 경우 OCR이 실패할 수 있습니다. 큰 글자 위주로 잘 보이는 부분을 캡쳐해 보세요.

サポート

使用中に困ったときは、まずよくある質問をご確認ください。

よくある質問

保存したスポットが見当たりません

次の2点をご確認ください。

  • 「行きたい」タブの右下にある リスト表示 ボタンを押しましたか?デフォルト画面は地図です。
  • Instagramから「共有」で保存した場合は、お店の名前を入力して初めて保存が完了します。

出口番号が表示されません

以下のケースでは出口案内が表示されないことがあります。

  • 保存スポットに座標がない場合(まれにInstagram保存直後) → 次回起動時に自動補完されます。
  • 到着駅がデータベース(約325駅)に含まれていない場合 → 一部の郊外駅はまだ未対応。ご連絡いただければ追加します。

表示言語を変えたい

地下鉄タブのウェルカム画面または駅名を長押しすると言語選択シートが開きます(한국어・日本語・English・中文)。

スクリーンショットから場所を抽出できません

画像に韓国語/日本語/英語/中国語のテキストが明確に含まれている必要があります。文字が小さすぎたりぼやけているとOCRが失敗することがあります。大きい文字がはっきり写っている部分をキャプチャしてください。

Frequently Asked Questions

I can't see my saved places

Check the following:

  • Did you tap the List button at the bottom-right of the Saved tab? The default view is the map.
  • If you saved via Instagram's share sheet, you need to enter the place name to finish saving.

The exit number isn't showing

The exit hint can be hidden in two cases:

  • The saved place has no GPS coordinates (rare, just after Instagram save) → the app fills these in on next launch.
  • The destination station isn't in the dataset (~325 stations). Some outer-Seoul stations are missing — let me know and I'll add yours.

How do I change the display language?

Long-press the welcome title or any station name on the Subway tab to open the language picker (한국어 · 日本語 · English · 中文).

Screenshot OCR isn't extracting place names

The image needs clearly readable Korean / Japanese / English / Chinese text. Tiny or blurry text can fail. Capture a tight crop where the place name is prominent.

Accessibility

KORA supports VoiceOver, Dynamic Type, and color-blind-friendly design. If you have accessibility feedback, please email me.
