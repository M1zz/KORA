개인정보 처리방침

최종 업데이트: 2026년 5월 31일

KORA(이하 "본 앱")는 사용자의 개인정보를 가능한 한 적게 수집하고, 수집한 정보는 모두 사용자의 기기에만 저장하는 것을 원칙으로 합니다. 본 앱에는 별도의 서버가 없으며, 어떤 사용자 데이터도 운영자에게 전송되지 않습니다.

1. 수집하는 정보

  • 저장한 장소 정보 (사용자가 직접 추가) — 이름, 주소, 좌표, 카테고리, 사진 URL, 가까운 역, 출구 번호, 메모/링크.
  • 위치 정보 — 앱 사용 중에만, 가장 가까운 지하철 역을 계산하기 위해 사용됩니다. 저장되거나 외부로 전송되지 않습니다.
  • 사진 — 사용자가 스크린샷 OCR 기능을 위해 직접 선택한 경우에만 일시적으로 접근합니다. 추출 후 즉시 폐기됩니다.
  • 클립보드 — 앱이 활성화될 때 URL을 자동 감지하기 위해 클립보드를 읽을 수 있습니다. 사용자가 명시적으로 추가하지 않는 한 저장되지 않습니다.

2. 정보 사용 방식

  • 수집된 모든 데이터는 사용자 기기 내부(iOS UserDefaults 및 App Group 컨테이너)에만 저장됩니다.
  • 운영자가 운영하는 서버는 존재하지 않으며, 사용자 데이터가 운영자에게 송신되는 일은 없습니다.
  • 광고, 분석(Analytics), 사용자 추적은 사용하지 않습니다.

3. 외부 서비스

장소 검색 및 사진 조회를 위해 다음 외부 API에 검색 키워드 등을 송신할 수 있습니다.

  • Kakao Local API — 장소명 검색, 좌표 조회, 공식 사진
  • Naver Search API — 보조 이미지 검색
  • Apple Maps — 외부 지도 앱으로 길 안내 호출

이들 서비스는 각 회사의 개인정보 정책에 따라 운영됩니다. 본 앱은 검색에 필요한 키워드와 좌표만 송신하며, 사용자 식별 정보는 전달하지 않습니다.

4. 데이터 보관 및 삭제

  • 저장된 장소 데이터는 앱을 삭제하면 모두 함께 삭제됩니다.
  • 앱 내에서 개별 장소를 직접 삭제할 수 있습니다.
  • iCloud 동기화는 사용하지 않으므로 데이터가 외부에 백업되지 않습니다 (iOS 기기 백업 시 디바이스 단위로 함께 백업됩니다).

5. 권한 사용 사유

  • 위치 (사용 중) — 현재 위치에서 가장 가까운 지하철 역 식별
  • 사진 라이브러리 — 사용자가 선택한 스크린샷에서 장소명 추출(OCR)

6. 만 14세 미만 어린이의 개인정보

본 앱은 만 14세 미만 어린이로부터 의도적으로 개인정보를 수집하지 않습니다.

7. 정책 변경

본 처리방침이 변경되는 경우, 이 페이지에 변경된 내용과 적용일을 게시합니다.

プライバシーポリシー

最終更新: 2026年5月31日

KORA(以下「本アプリ」)は、ユーザーの個人情報を必要最小限のみ取り扱い、 取得した情報はすべてユーザー端末のみに保存することを原則とします。 本アプリには専用サーバーが存在せず、ユーザーデータが運営者へ送信されることはありません。

1. 取得する情報

  • 保存スポット情報(ユーザー操作で追加)— 名称、住所、座標、カテゴリ、写真URL、最寄り駅、出口番号、メモ/リンク。
  • 位置情報 — アプリ使用中のみ、最寄り駅の算出に使用します。保存・外部送信はしません。
  • 写真 — スクリーンショットOCR機能のためにユーザーが選択した時のみ、一時的にアクセスします。テキスト抽出後すぐに破棄されます。
  • クリップボード — アプリがアクティブになる際にURLを自動検出するため読み取ることがあります。ユーザーが明示的に追加しない限り保存されません。

2. 情報の利用方法

  • 取得した全データはユーザー端末内(iOS UserDefaults および App Group コンテナ)にのみ保存されます。
  • 運営者が管理するサーバーは存在せず、ユーザーデータが運営者に送られることはありません。
  • 広告、分析(Analytics)、ユーザートラッキングは行いません。

3. 外部サービス

スポット検索や画像取得のため、以下の外部APIに検索キーワードなどを送信することがあります。

  • Kakao Local API — スポット名検索、座標取得、公式写真
  • Naver Search API — 補助的な画像検索
  • Apple Maps — 外部マップアプリでの経路案内呼び出し

これらのサービスは各社のプライバシーポリシーに従って運用されます。本アプリは検索に必要なキーワードと座標のみを送信し、ユーザー識別情報は送りません。

4. データの保存と削除

  • 保存スポットデータはアプリを削除すると共に消去されます。
  • アプリ内で個々のスポットを削除できます。
  • iCloud同期は使用していないため、外部にバックアップされることはありません(iOSのデバイスバックアップでは端末単位で含まれます)。

5. 権限の使用理由

  • 位置情報(使用中) — 現在地から最寄りの地下鉄駅を特定
  • 写真ライブラリ — ユーザーが選択したスクリーンショットからスポット名を抽出(OCR)

6. 14歳未満の子供の個人情報

本アプリは14歳未満の子供から意図的に個人情報を取得することはありません。

7. ポリシーの変更

本ポリシーが変更された場合、本ページに変更内容と適用日を掲載します。

Privacy Policy

Last updated: May 31, 2026

KORA ("the App") collects as little personal information as possible and keeps everything it does collect on your device. The App has no backend server, and no user data is ever transmitted to the developer.

1. Information We Collect

  • Saved place data (added by you) — name, address, coordinates, category, photo URLs, nearest station, exit number, link/note.
  • Location — accessed only while the App is in use, to find the nearest subway station. It is not stored or transmitted.
  • Photos — accessed only when you explicitly pick a screenshot for OCR. The image is processed for text extraction and then discarded.
  • Clipboard — read when the App becomes active so it can detect a URL automatically. Nothing is stored unless you explicitly add it.

2. How We Use Information

  • All collected data stays on your device only (iOS UserDefaults and an App Group container).
  • There is no developer-operated backend. No user data is sent to the developer.
  • No advertising, no analytics, no user tracking.

3. Third-party Services

For place search and image retrieval, the App may send a search query to:

  • Kakao Local API — place search, coordinate lookup, official photos
  • Naver Search API — supplementary image search
  • Apple Maps — external map app handoff for directions

These services operate under their own privacy policies. The App sends only the search keyword and coordinates required, no user-identifying information.

4. Storage and Deletion

  • Saved places are deleted when you remove the App.
  • You can delete individual places from inside the App.
  • The App does not use iCloud sync, so your data isn't backed up externally (iOS device-level backup will include it as part of the device).

5. Permissions and Why

  • Location (when-in-use) — to identify the nearest subway station to you.
  • Photo Library — to OCR a screenshot you pick for place-name extraction.

6. Children's Privacy

The App does not knowingly collect personal information from children under 14.

7. Changes to This Policy

If this policy changes, the updated version and its effective date will be posted on this page.
